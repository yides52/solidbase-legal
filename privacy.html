Privacy Policy

Last updated: July 8, 2026

This Privacy Policy explains how the Solid Base Estimator application ("the App") handles data. The App is an internal tool operated by Solid Base ("we", "us", "our") and is not distributed publicly.

1. Data We Access

When you connect the App to QuickBooks Online, we access the following data on your behalf:

  • Customers list: to identify which customer an estimate should be linked to.
  • Items list: to map parsed estimate line items to QuickBooks items.
  • Estimate creation: we write new estimates to QuickBooks Online.

We do not read invoices, bills, transactions, banking data, or any other QuickBooks data.

2. Data You Provide

The App processes the estimate text you paste into it. This text is sent to the Anthropic API for parsing. Only the pasted estimate text is transmitted — no QuickBooks customer names, item lists, or other data is sent to Anthropic.

3. Where Data Is Stored

The App runs locally on the user's own computer. It stores:

  • An Anthropic API key (locally, on the user's machine, under the user's Windows profile)
  • QuickBooks OAuth access and refresh tokens (locally, same location)
  • A cached list of QuickBooks items (locally, same location)

No data is transmitted to any Solid Base server. There is no Solid Base server.

4. Third Parties

We use the following third-party services, each governed by its own privacy policy:

  • Intuit QuickBooks Online — for reading and writing accounting data (privacy policy)
  • Anthropic — for AI parsing of estimate text (privacy policy)

5. Data Retention

The App itself does not retain estimate text after it has been parsed. Tokens and cached items are retained until the user disconnects or deletes the local settings file.

6. Security

OAuth tokens are stored on the user's local machine. Anyone with access to that machine could access the tokens. We recommend using standard OS-level protections (login password, disk encryption) on the machine that runs the App.

7. Your Choices

You can disconnect QuickBooks Online at any time from within the App's Settings screen. This immediately clears all stored tokens.

8. Changes to This Policy

We may update this Policy from time to time. Any changes will be reflected in the "Last updated" date at the top.

9. Contact

For privacy questions, please contact Solid Base.
