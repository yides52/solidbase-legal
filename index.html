Solid Base Estimator

Internal estimate-to-QuickBooks automation for Solid Base construction.

What it does

Solid Base Estimator is an internal desktop application that takes construction estimates in plain text form and automatically creates matching estimates in QuickBooks Online. It's built for use by Solid Base to reduce manual data entry between project proposals and accounting.

How it works

  • User pastes an estimate description into the app
  • Emily (an AI assistant powered by the Anthropic API) parses the text into structured line items
  • The user reviews the parsed rows on-screen
  • With one click, the estimate is pushed into QuickBooks Online for the linked customer

QuickBooks integration

The application connects to QuickBooks Online through Intuit's official OAuth 2.0 flow. It only writes estimates and reads item/customer lists needed to build those estimates. No other QuickBooks data is accessed.

Solid Base Estimator is a private in-house tool. It is not distributed publicly.
